1. cd server  
2. pip install pypiserver   
3. mkdir ~/packages  
4. Check the actual path (windows): Resolve-Path ~/packages 
   i.e: C:\Users\Ian\packages
5. copy your package to /server/packages (both .tar.gz and .whl????)
6. pypi-server run -p 3001 ~/packages
7. from the client run the following command:
   pip install --extra-index-url http://localhost:3001/simple/ math_operations
